# learning-DynamoDB

🐞🐞🐞 AWS DynamoDBについて学習するためのプログラム。  

## 実行方法

`terraform`ディレクトリに移動して、以下のコマンドを実行します。  

```shell
terraform init
terraform plan
terraform apply
```

これで、DynamoDBのテーブルが作成されます。  

---

次に、`app`ディレクトリに移動して、以下のコマンドを実行します。  

```shell
yarn dev
```

これで、作成したDynamoDBのテーブルに対して、サンプル処理が実行されます。  

## セキュリティ

DynamoDBはVPC(サブネット)に属することができないため、セキュリティグループを設定することができません。  
これはIPアドレスによるアクセス制限をかけることができないことを意味します。  

DynamoDBのアクセス制御はアクセスキーとシークレットキーによって行われます。  
したがって、これらの管理は厳重に行う必要があります。  

自分的にはデータベースに対するアクセス制御は「IPアドレス制限」と「トークン(ID/PW・アクセスキー)」の2要素で行いたいため、DynamoDBは少し不安です、、、  
間違ってもアクセスキーとシークレットキーをGitHubに上げてしまわないように注意してください。  

原則として環境変数を使用するため、間違って漏洩することはありませんが、それでも細心の注意を払ってください。
